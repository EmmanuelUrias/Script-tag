In JavaScript, global objects are objects that are available in the global scope. This means that you can access these objects from any script or function in your code, without having to explicitly declare or import them.

There are several built-in global objects in JavaScript, including:

Math: This object provides mathematical functions and properties, such as Math.PI and Math.sin().

Date: This object provides functions for working with dates and times.

String: This object provides functions for working with strings, such as
String.prototype.substr() and String.prototype.toLowerCase().

Array: This object provides functions for working with arrays, such as
Array.prototype.push() and Array.prototype.sort().

In addition to these built-in global objects, you can also create your own global objects by declaring variables in the global scope. For example:

var myGlobalObject = {};

Keep in mind that using global objects can lead to conflicts and namespace pollution, so it's generally a good idea to avoid creating global objects unless absolutely necessary. Instead, consider using modules or other techniques to manage your code.
