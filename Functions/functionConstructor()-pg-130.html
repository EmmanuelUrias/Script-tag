Function constructor:

Another example of a constructor function:
function Car(make, model, year) {
this.make = make;
this.model = model;
this.year = year;
}

To use this constructor function you'd have to create a new car object:
let myCar = new Car("Toyota", "Camry", 2020);

This would create a new Car object with the properties make, model, and year set to the values "Toyota", "Camry", and 2020, respectively.

Constructor functions are typically used to create multiple objects with the same properties and methods, but with different initial values. They are a useful way to encapsulate data and behavior and to create reusable code.
