Learning JavaScript Arrays

When looking at the code it looks like the carsTextDisplay variable should be an empty string but when we declare it outside the for loop and update it within the for loop it gets it's new value and we can display the correct data to the page

You could define the carsTextDisplay variable outside of the loop and update the innerHTML property once the loop has finished executing, like this:
let carsTextDisplay = "";
for (let i = 0; i < carsforsale.length; i++) {
carsTextDisplay += carsforsale[i] + " "; }
document.getElementById("array-demo").innerHTML = carsTextDisplay;
