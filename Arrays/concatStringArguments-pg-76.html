JavaScript Array concat()

The concat() method follows the merging pattern:

What is an argument?:
An argument is a value that is passed to a function when it is called. ex.
function add(x, y) {
return x + y;
}
console.log(add(1, 2)); // 3

Well isn't that a parameter?:
A parameter is a variable in a function definition that represents an argument passed to the function. ex.
function add(x, y) {
return x + y;
}
console.log(add(1, 2)); // 3
console.log(add(3, 4)); // 7

Although they look extremely similar here is a key difference:
a parameter is a placeholder for an argument that is defined when the function is defined.
An argument, on the other hand, is a value that is passed to a function when it is called. When a function is called with arguments, the arguments are matched up with the corresponding parameters and used to perform a specific task or calculation.
