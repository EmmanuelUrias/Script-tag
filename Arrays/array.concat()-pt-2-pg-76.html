JavaScript Array concat()

The concat() method follows the merging pattern:

difference between .forEach and .map:
let arr = [1, 2, 3, 4, 5];
// Using forEach()
arr.forEach(item => console.log(item * 2));
// Output: 2 4 6 8 10
// Using map()
let newArr = arr.map(item => item * 2);
console.log(newArr); // [2, 4, 6, 8, 10]
