These are JavaScript For In Loops

I will show how the for/in statement loop through and over Array values.

The for/in loop is special because you can pair it up with an array and it will loop through the properties of the array just like it did with the properties of the object.

You should drop the idea of using for in over the array loops if you are keen on the order of the index. It is important and better for you to use a for of loop, for loop, and the Array.forEach() whenever you think the order is special.
